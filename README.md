# Corrida 3D - jogo2

Jogo de corrida 3D em navegador com Three.js, fisica simples de carro, camera dinamica em terceira pessoa e velocimetro.

Se voce esta vendo somente esta pagina branca do README, o jogo nao foi aberto ainda.
Abra o arquivo `index.html` por um servidor local.

## Como rodar

Na pasta do projeto, rode:

```bash
npm start
```

Depois acesse:

```text
http://127.0.0.1:5173
```

Controles:

- `W` ou `Seta para cima`: acelerar
- `S` ou `Seta para baixo`: frear / marcha re
- `A` / `D` ou `Setas esquerda/direita`: virar

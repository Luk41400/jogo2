# Corrida 3D

Jogo de corrida 3D em navegador com Three.js, física simples de carro, câmera dinâmica em terceira pessoa e velocímetro.

## Como rodar

Sirva a pasta do projeto com qualquer servidor estático e abra o navegador:

```bash
python -m http.server 5173
```

Depois acesse:

```text
http://127.0.0.1:5173
```

Controles:

- `W` ou `Seta para cima`: acelerar
- `S` ou `Seta para baixo`: frear / marcha ré
- `A` / `D` ou `Setas esquerda/direita`: virar
